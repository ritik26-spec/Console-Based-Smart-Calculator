# Smart Calculator 🧮

A simple Python-based Smart Calculator built using **Python** and **NumPy**.  
This calculator performs multiple mathematical operations on multiple numbers with user-friendly input handling.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Power Calculation
- Modulus Operation
- Multiple number input support
- Error handling for invalid input
- Division by zero protection
- Interactive command-line interface

---

## 🛠️ Technologies Used

- Python
- NumPy
